# Installation

## Stable release

To install EnsPrecipPP, run this command in your terminal:

```sh
uv add ensprecippp
```

Or if you prefer to use `pip`:

```sh
pip install ensprecippp
```

## From source

The source files for EnsPrecipPP can be downloaded from the [Github repo](https://github.com/Curallin/ensprecippp).

You can either clone the public repository:

```sh
git clone git://github.com/Curallin/ensprecippp
```

Or download the [tarball](https://github.com/Curallin/ensprecippp/tarball/master):

```sh
curl -OJL https://github.com/Curallin/ensprecippp/tarball/master
```

Once you have a copy of the source, you can install it with:

```sh
cd ensprecippp
uv pip install .
```
